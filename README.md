# 🎓 Student Management System

A Java Swing desktop application for managing student records with MySQL database integration.

## 📌 Overview

The Student Management System is a desktop-based application developed using Java Swing and JDBC.

It allows users to manage student information through a graphical user interface while storing data in a MySQL database.

---

## 🚀 Features

✅ Add new students  
✅ View student records  
✅ Search students  
✅ Update student information  
✅ Delete student records  
✅ MySQL database integration  
✅ User-friendly GUI interface

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/felixtare585-gif/Student-Management-System-Java.java.git
```

### Open the project

Open the project using:

- VS Code
- IntelliJ IDEA
- Eclipse

---

## 🗄️ Database Setup

1. Install MySQL.
2. Create a database named:

```sql
student_management
```

3. Import the SQL file (if available).

4. Update your database credentials inside the project.

---

## ▶️ Running the Project

Compile and run the Java files.

Make sure:

- Java JDK is installed
- MySQL Server is running
- JDBC Connector is added
